# Linear A Decipherment Research: Most Promising Results

*Last updated: 2026-03-18 | Research sessions completed: 5 | Corpus: 1721 inscriptions, 52 sites*

This project applies systematic computational and linguistic methods to Linear A, the undeciphered writing system of Minoan Crete (~1800–1450 BCE). All claims are hypotheses with stated confidence levels — nothing here constitutes a decipherment.

## The IOZa2 Liturgical Inscription — Best Structural Finding

The stone vessel inscription **IOZa2 from Iouktas** (a Minoan peak sanctuary) contains the most fully-interpreted sequence yet identified. Analysis across all 1721 inscriptions confirms it instantiates a **7-element liturgical template** attested at multiple sites:

```
[HEADER]    A-TA-I-*301-WA-JA     ← ritual opener (confirmed at 5 sites, 11 inscriptions)
[SLOT 1]    JA-DI-KI-TU           ← DI-KI element (see below)
[ELEMENT 1] JA-SA-SA-RA-ME        ← divine name root JA-SA-SA-RA + suffix -ME
[ELEMENT 2] U-NA-KA-NA-SI         ← locked companion to JA-SA-SA-RA-ME (4/4 co-occurrences)
[ELEMENT 3] I-PI-NA-MA            ← ritual term (4 sites)
[ELEMENT 4] SI-RU-TE              ← closing ritual term (multiple sites)
[CODA]      TA-NA-RA-TE-U-TI-NU  ← Iouktas-specific coda (dedicant/sanctuary marker?)
```

The full IOZa2 text (linear transliteration, conventional values):
`A-TA-I-*301-WA-JA 𐄁 JA-DI-KI-TU 𐄁 JA-SA-SA-RA-ME 𐄁 U-NA-KA-NA-SI 𐄁 I-PI-NA-MA 𐄁 SI-RU-TE 𐄁 TA-NA-RA-TE-U-TI-NU 𐄁 I 𐄁`

**Multi-site validation** of the template:
- IOZa2 (Iouktas): all 7 elements ✓
- PKZa11 (Palaikastro): 6/7 elements ✓ (DI-KI = A-DI-KI-TE-TE, divine name = SA-SA-RA-ME)
- PKZa12 (Palaikastro): header + DI-KI + divine name ✓
- TLZa1 (Troullos): header + divine name + elements 2–4 ✓
- KOZa1 (Kophinas): elements 2–4 ✓

**Confidence: Medium-High (Tier 3, Formulaic)**

## The DI-KI Element — Possible Sacred Toponym

The DI-KI syllable pair (conventional transliteration) appears in **9 corpus occurrences** across 4 sites with **7.4× ritual enrichment** (67% ritual rate vs. 9% baseline).

**Most significant finding:** DI-KI occupies the same structural position — slot 1 of the liturgical sequence, between the ritual header and the divine name — at BOTH Iouktas and Palaikastro:

- Iouktas: `JA-DI-KI-TU` (IOZa2, slot 1)
- Palaikastro: `A-DI-KI-TE` / `A-DI-KI-TE-TE` (PKZa12, PKZa11, slot 1)

This makes DI-KI an **integral structural element** of the ritual formula, not a site-specific variant. The conventional transliteration DI-KI is phonologically compatible with Mt. Dikte/Dikti (Δίκτη), the sacred Cretan mountain associated in Greek mythology with divine birth — a concept plausibly inherited from Minoan religion.

**Administrative uses** at Haghia Triada show DI-KI-SE appearing in personal name lists — consistent with DI-KI being a recognized toponym that gave rise to personal names.

**Confidence: Low-Medium (Tier 4, Lexical)**

## Grammatical Suffix Evidence

### -JA as Absolutive Marker (Medium-High Confidence)

The most robust grammatical finding:

| Evidence | Result |
|----------|--------|
| KI-RO co-occurrence rate | **0.0%** across 58 records, 17 sites |
| Word-initial frequency | 59% of -JA tokens appear word-initial in inscriptions |
| Ritual enrichment | 2.25× ritual enrichment |
| Hurrian alignment score | 4.0/5.0 (absolutive) |

Zero KI-RO co-occurrence across 17 sites for -JA words is the strongest single negative evidence in the corpus: words ending in -JA never appear in deficit-accounting contexts. This is consistent with -JA being a count-noun marker or absolutive case suffix used for commodity recipients/entities (not deficits or transactions).

## The JA-SA-SA-RA Divine Name Root

Multiple attested inflected forms confirm JA-SA-SA-RA is a divine name root:

| Form | Occurrences | Register |
|------|-------------|---------|
| JA-SA-SA-RA-ME | 7 | Ritual only (Iouktas, Palaikastro, Platanos, Troullos, Psykhro) |
| SA-SA-RA-ME | 2 | Ritual only |
| JA-SA-SA-RA | 1 | Ritual |
| JA-SA-SA-RA-MA | 1 | Ritual |
| A-SA-SA-RA-ME | 1 | Ritual (Clay vessel, Iouktas) |
| A-SA-SA-*802-ME | 1 | Ritual (Prassa) |

The bare root, -ME form, and -MA form are all attested, consistent with a divine name inflected for at least two grammatical categories. JA-SA-SA-RA-ME **always precedes** U-NA-KA-NA-SI when they co-occur (4/4 inscriptions across 3 sites).

**Confidence: Medium-High (Tier 3, Formulaic)**

## Language Family — Current Evidence State

No language family identification has been achieved. Current evidence is compatible with Hurrian (Tier 5, Low confidence) but is not definitive.

| Language Family | Best Evidence | Problem |
|----------------|---------------|---------|
| **Hurrian** | 14 high-confidence suffix alignments (≥4.0/5.0); -JA absolutive alignment; -TE directive alignment | Compatibility ≠ relatedness; Luwian test (0 high-conf alignments) favors Hurrian only marginally |
| **Luwian/Anatolian** | Woudhuizen debate (not resolved) | 0 high-confidence morpheme alignments in systematic test |
| **Proto-Semitic** | N-gram fingerprinting ranked it first (but method was contaminated) | N-gram test inconclusive; methodological problems |
| **Isolate** | Most parsimonious given failures | Does not explain Hurrian suffix compatibility |

The language appears to be **agglutinative** (Brent Davis 2014; confirmed by suffix paradigm structure). Ergative or active-stative alignment is possible but not proven.

## Ritual Register Structure

Stone vessel and metal object inscriptions form a **structurally distinct corpus** from administrative tablets. Key confirmed facts:

- **10 tokens appear exclusively in ritual contexts** (100% ritual, ≥3 occurrences)
- **A-TA-I-*301-WA-JA** is a ritual header: 11 occurrences, 5 sites, 100% ritual, 10/11 inscription-initial
- **JA-SA-SA-RA-ME**: 5 sites, 100% ritual, 5.13× enrichment for -ME suffix as a class
- **-U-TI-NU** terminal: Iouktas-site-specific (3 types, all Iouktas stone vessels)

The ritual register appears to be a separate linguistic/formulaic system from the administrative register — possibly a liturgical dialect or dedicated ritual vocabulary.

## Active Research Questions (Priority Order)

1. **What is U-NA-KA-NA-SI?** Locked companion to JA-SA-SA-RA-ME in 4/4 co-occurrences. Most frequently recurring unknown ritual term after the divine name itself.
2. **What is sign *301?** Appears in ritual header formula and as standalone sign on 230+ HT administrative nodules — dual use suggests it may be polyfunctional (logographic + phonetic). Highest-priority paleographic identification needed.
3. **What does the -TU vs -TE contrast on DI-KI mean?** JA-DI-KI-TU (Iouktas) vs A-DI-KI-TE (Palaikastro) — could reveal the Minoan case system if DI-KI is indeed a toponym.
4. **Is -U-TI-NU a sanctuary identifier for Iouktas?** If so, it may appear in Linear B records of goods sent to/from Iouktas.
5. **Does DI-KI appear in Linear B Knossos tablets?** Cross-script comparison could confirm or refute the sacred toponym hypothesis.
6. **What is the grammatical function of -ME?** 5.13× ritual enrichment; exclusive divine-name attachment; positional analysis inconclusive. Next test: check whether bare JA-SA-SA-RA vs -ME form differs by inscription type, site, or structural position.
7. **Can the Proto-Semitic hypothesis be constrained and tested?** N-gram analysis ranked it first (contaminated method) — needs clean re-test with triconsonantal root comparison.
