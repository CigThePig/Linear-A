## Administrative Closing Formulas

### KU-RO — "Total"

| Property | Value |
|----------|-------|
| Sequence | KU-RO |
| Proposed meaning | "total" (accounting grand total) |
| Confidence | High |
| Evidence tier | Tier 3 (Formulaic) |
| Corpus occurrences | 39 bigram occurrences |
| Basis | Positional analysis: KU-RO consistently appears at the end of accounting sequences after commodity+quantity pairs, in the same structural position as Linear B *to-so* ("so much / total") |
| Cross-site attestation | Haghia Triada (dominant), Khania, Phaistos, and others |
| First documented | Long-established in scholarship (Godart, Olivier, et al.) |

**Structural pattern:**
```
[entity A] [logogram] [number]
[entity B] [logogram] [number]
[entity C] [logogram] [number]
KU-RO [total number]
```

### KI-RO — "Owed / Deficit"

| Property | Value |
|----------|-------|
| Sequence | KI-RO |
| Proposed meaning | "owed" or "deficit" (accounting shortfall) |
| Confidence | High |
| Evidence tier | Tier 3 (Formulaic) |
| Corpus occurrences | ~17 bigram occurrences |
| Basis | Positional analysis: KI-RO appears in deficit/outstanding balance contexts, contrasted with KU-RO (total received/allocated). Parallel to Linear B accounting conventions for outstanding debts. |
| Cross-site attestation | Haghia Triada primarily; attested at other palatial sites |
| First documented | Established in scholarship |

**Structural pattern:**
```
[total] KU-RO [number]
[deficit/owed] KI-RO [number]
```

| -RO | 78 | Appears in KI-RO and many other tokens; possible case or functional marker |
| -JA | 65 | Common across administrative and non-administrative contexts |
| -RE | 58 | Possibly a different grammatical category from -RO despite phonological similarity |
| -TE | 56 | May be a personal name suffix or grammatical case marker |
| -NA | 54 | Possibly genitive or locative; cf. Hurrian locative -na |
| -TI | 49 | Common across multiple support types |

**Note:** These are counts from the corpus distributional analysis. The grammatical interpretation of each suffix is a Tier 5 hypothesis, not confirmed fact. The frequency data itself is Tier 1 (structural).
