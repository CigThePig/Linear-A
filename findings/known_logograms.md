# Confirmed Commodity Logograms

This file documents logograms in Linear A with high-confidence identifications. All entries are based on Linear B parallels, archaeological context, and/or positional analysis in the corpus. Do not add speculative entries here.

## Confirmed Logograms

### GRA — Grain / Cereal

| Property | Value |
|----------|-------|
| Sign label | GRA |
| Commodity | Grain / cereal crop |
| Confidence | High |
| Evidence tier | Tier 2 (Logographic) |
| Basis | Linear B *GRA* logogram (same sign, same administrative context); appears before quantities in inventory contexts at multiple sites |
| Attested sites | Haghia Triada, Phaistos, Knossos, and others |
