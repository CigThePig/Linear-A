# Linear A Field State — External Research Notes
*Compiled: 2026-03-22*

This file summarizes the current scholarly consensus on the state of Linear A decipherment research — barriers, language family debates, key researchers, and what a genuine breakthrough would require. It draws on web-accessible sources gathered in March 2026.

---

## Current Status: Still Undeciphered

Linear A remains completely undeciphered as of early 2026. Signs can be read aloud using approximate phonetic values borrowed from Linear B, but the words produced are meaningless because the underlying Minoan language is unknown. More than 125 years of scholarship, and several years of AI-assisted analysis, have not produced a verified reading.

**Corpus facts (external consensus):**
- ~1,427 surviving specimens totaling ~7,362–7,396 signs
- Our project corpus contains 1,721 records (including fragmentary inscriptions)
- Mathematicians estimate 10,000–12,000 sign examples are needed for statistical decipherment; the current corpus is below this threshold
- ~6,000 Linear B sign examples exist for comparison — roughly 3× the Linear A corpus

---

## Core Barriers

1. **Small corpus** — Only ~1,400–1,427 inscriptions survive (vs. 6,000+ for Linear B). Sample sizes are too small to reliably detect all phonotactic and morphological patterns.

2. **No bilingual text** — No inscription records the same content in Linear A and a known language. Without a Minoan "Rosetta Stone," external anchoring is impossible. All proposed vocabulary is inferred from positional/distributional analysis.

3. **Unknown language family** — Scholarly consensus in early 2026 treats Minoan as a probable language **isolate** — no known living or dead relatives. This removes the comparative linguistic framework that allowed decipherments of Mycenaean Greek (Linear B), Ugaritic, and Akkadian cuneiform.

4. **Phonetic values are hypotheses** — The ~60–80% sign overlap with Linear B allows conventional phonetic labels to be borrowed, but those labels are not demonstrated readings of Minoan. They are working hypotheses.

## Language Family: Scholarly Debates

| Hypothesis | Strongest Evidence | Main Objection |
|------------|-------------------|----------------|
| **Language isolate** | Most parsimonious; no strong genetic signal | Does not explain structural similarities to known language types |
| **Hurrian** | Agglutinative morphology; some suffix parallels | Temporal and geographic gap; compatibility ≠ relatedness |
| **Luwian/Anatolian** | Vocabulary-level matches (Nepal & Perono Cacciafoco 2024); Woudhuizen argument | Matches may be loanwords; 0 grammar-level alignments in systematic tests |
| **Proto-Semitic** | Cyrus Gordon's original hypothesis; some phonological arguments | Methodologically weak; no triconsonantal root match at scale |
| **Proto-Indo-European** | Some researchers (Woudhuizen) argue Minoan is an archaic IE language | Duhoux: heavy affixation makes IE affiliation "unlikely"; no systematic evidence |
| **Uralic/Etruscan** | Older fringe proposals | No serious current scholarly support |

**Key researcher position:** Yves Duhoux (Univ. of Louvain, emeritus) demonstrated through structural analysis that Minoan makes heavy use of prefixes and suffixes attached to word stems — a morphological pattern that makes an Indo-European affiliation unlikely.

**Implication for this project:** Hurrian remains the best-supported structural comparison based on our own morpheme alignment work (14 high-confidence alignments, vs. 0 for Luwian). However, Luwian vocabulary matches from external studies may suggest loanwords from Anatolian trade contact — consistent with Minoan's isolate status while still showing lexical borrowing.

## What Would Constitute a Breakthrough

Every specialist working on Linear A agrees on a single critical point: **the breakthrough, if it comes, will come from the ground — not from better algorithms.**

Specifically, a breakthrough requires:

1. **A bilingual inscription** — An object with the same content in Linear A and a known language (Egyptian, Hittite, Akkadian, Mycenaean Greek). Ground-penetrating radar surveys at Knossos, Phaistos, and Malia have identified unexcavated structures potentially containing administrative quarters with clay tablet deposits.

2. **A substantially larger archive** — Pushing the sign count from ~7,362 to the ~10,000–12,000 threshold mathematicians suggest is needed. The 2025 Knossos ivory scepter adds 119 signs; continued excavation of palace complexes is the most likely path.

3. **A proven linguistic relative** — If a language genetically related to Minoan is found with more text, the shared vocabulary would allow cross-language anchoring even without a bilingual text.

Current AI and computational methods are organizing the evidence more systematically and generating testable structural hypotheses, but without new primary data they cannot overcome the fundamental information deficit.
